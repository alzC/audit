Accessibilité

Mauvaise lisibilité

problème : Vérifier que les contrastes entre le premier plan et l’arrière-plan rencontrent les seuils de contrastes exigés par les WCAG 2 AA
la plupart des elements textuels on un contraste de couleurs insuffisant de 1.76 (couleur d’avant plan : #ffb536, couleur d’arrière plan : #ffffff, taille de police : 24.0pt (32px), graisse : normal). Contraste de couleur attendu : 3:1

Niveau de titre

problème : Dans plusieurs pages du site, les niveaux de titre sont "sautés". Par exemple, dans la home page il n'y a pas de titre de niveau 1. Cela pose un problème d'accessibilité étant donné que les utilisateurs utilisant des lecteurs d'écran utilisent les niveaux de titres pour se repérer dans une page. L'absence d'un des niveaux peut donc amener de l'incompréhension chez l'utilisateur.
